# Windows Boot Performance Diagnostic Toolkit

A read-only PowerShell toolkit for Windows startup and boot-performance analysis.

## Features

- Current uptime and last boot time
- Boot and shutdown event review
- Startup application inventory
- CSV, JSON, and HTML reports

## Run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Windows_Boot_Performance_Diagnostic_Toolkit.ps1
```

## Safety

Read-only reporting only. No startup settings are changed.
